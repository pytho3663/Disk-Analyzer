# 磁盘空间分析器 v2.0 - Windows一键安装版

🚀 **现代化磁盘空间管理工具** - 一键安装，立即使用！

## 📥 一键安装

### 方式1：快速安装（推荐）
1. 下载 `DiskAnalyzer-Setup.exe`
2. 双击运行安装程序
3. 桌面自动生成快捷方式
4. 点击即可使用！

### 方式2：绿色版（无需安装）
1. 下载 `DiskAnalyzer-Portable.zip`
2. 解压到任意目录
3. 双击 `run.bat` 即可运行

## ✨ 核心功能

### 🎯 智能扫描
- ⚡ **直接扫描**：无需等待预扫描，立即开始

- 🗑️ **安全删除**：防止误删系统文件
